# CodePush 版本控制

|表达式|能接受更新的版本|
|-----|---|
|1.2.3|只有 1.2.3|
|1.2.x|所有 1.2.|
|1.2.3 - 1.2.7|版本号在 1.2.3(含) 到 1.2.7(含)|
|>=1.2.3 <1.2.7|版本号在 1.2.3(含) 到 1.2.7(不含)|
|~1.2.3|等价于 >=1.2.3 <1.3.0|
|^1.2.3|等价于 >=1.2.3 <2.0.0|
|*|所有|
